# 分身协作架构

## 核心理念
分身(qwen2:7b)是我的算力扩展，用于：
- 并行处理任务
- 批量生成内容
- 快速迭代优化
- 减轻主模型负担

## 协作模式

### 1. 并行任务分发
```
主模型(我) ──┬── 复杂决策、策略制定、创意方向
            │
分身(qwen2) ─┴── 批量生成、数据处理、简单重复任务
```

### 2. 实时协作场景

| 场景 | 主模型负责 | 分身负责 |
|------|-----------|---------|
| 短视频 | 创意方向、质量评估 | 批量生成主题/场景/文案 |
| 股票交易 | 买卖决策、风险控制 | 行情分析、新闻摘要 |
| 代码开发 | 架构设计、代码审查 | 生成代码片段、写测试 |
| 数据分析 | 结论输出、策略建议 | 数据清洗、统计计算 |
| 内容创作 | 定调、审核 | 批量生成变体 |

### 3. 分身API调用

```bash
# 基础调用
curl -s http://localhost:11434/api/generate -d '{
  "model": "qwen2:7b",
  "prompt": "任务内容",
  "stream": false
}'
```
