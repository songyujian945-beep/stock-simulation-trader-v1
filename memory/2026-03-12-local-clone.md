# 本地分身协作方案

## 分身配置
- **模型**: Qwen2:7B (7.6B参数, Q4_0量化)
- **位置**: http://localhost:11434
- **大小**: 4.4 GB
- **响应速度**: 10-15秒（首次加载约30秒）

---

## 协作模式

### 模式1: 任务分流
| 任务类型 | 分配给 | 原因 |
|---------|--------|------|
| 简单问答 | 本地分身 | 快速、离线可用 |
| 文案生成 | 本地分身 | 擅长创意内容 |
| 代码片段 | 本地分身 | 质量足够 |
| 复杂推理 | 云端我 | 需要更强能力 |
| 工具调用 | 云端我 | 分身无权限 |
| 文件操作 | 云端我 | 分身无访问 |

### 模式2: 批量生产
本地分身适合批量生成：
- 短视频标题（每次20-50个）
- 场景描述（每次10-20个）
- 文案变体（每次5-10个）
- 主题创意（每次5-10个）

### 模式3: 离线备用
网络故障时，自动切换到本地分身处理简单任务


```
云端我（复杂）→ 日程管理、邮件处理、决策建议

```

## 分身调用方式

### 方式1: curl直接调用
```bash
curl http://localhost:11434/api/generate -d '{
  "model": "qwen2:7b",
  "prompt": "你的问题或任务",
  "stream": false
}'
```

### 方式2: Python脚本
```python
import requests
response = requests.post(
    "http://localhost:11434/api/chat",
    json={
        "model": "qwen2:7b",
        "messages": [{"role": "user", "content": "问题"}]
    }
)
```

### 方式3: ollama命令
```bash
ollama run qwen2:7b "你的问题"
```

## 协作示例

### 示例1: 短视频标题生成
```bash
# 让分身生成标题
curl -s http://localhost:11434/api/generate -d '{
  "model": "qwen2:7b",
  "prompt": "生成10个抖音短视频标题，主题：程序员加班",
  "stream": false
}'
```

### 示例2: 批量场景生成
```bash
# 让分身为所有主题生成场景
for theme in "职场吐槽" "扎心真相" "情感共鸣"; do
  curl -s http://localhost:11434/api/generate -d "{
    \"model\": \"qwen2:7b\",
    \"prompt\": \"为'$theme'主题生成5个短视频场景\",
    \"stream\": false
  }"
done
```

## 分身能力评估

### ✅ 擅长
1. 中文表达（流畅、地道）
2. 创意生成（标题、文案）
3. 简单问答
4. 格式化输出（JSON、列表）
5. 内容扩展（给定主题生成内容）

### ⚠️ 一般
1. 复杂推理
2. 长文本生成（会重复）
3. 代码复杂度（适合简单片段）

### ❌ 不擅长
1. 工具调用
2. 文件操作
3. 实时信息
4. 长期记忆
